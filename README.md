# Configurations

## macOS

- `brew install pyenv fzf nvm trash gh`
- `brew install --cask tableplus postman docker`
- Softwares: VSCode, Sublime Text, iTerm2, Warp
- Disable accent menu: `defaults write -g ApplePressAndHoldEnabled -bool false`
